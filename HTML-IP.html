Q.5): Difference between block and inline?

Block Level Element Inline Element
A block level element starts in a new line. An inline element does not start with the new line
The div element is a block level andn is often used a a container for other HTML elements. The span element is an inline container used to mark up a part of a text, or a part of a document.

  1. Attributes adjusts the behaviour of an element like color, size, etc.
  2. Attributes can be returned in the lowercase letters only because they are case sensitive.
  3. Atributes provides additional information about elements.
